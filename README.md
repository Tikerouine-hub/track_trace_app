# Application Track & Trace

Projet personnel réalisé par Amine Tikerouine  
Étudiant en 3ᵉ année Bachelor Ingénierie Web

## Description

Application web de traçabilité industrielle permettant la gestion de lots, le suivi des statuts et l’historique des événements.

## Fonctionnalités

- Création de lots
- Mise à jour des statuts
- Historique de traçabilité
- Interface web dynamique

## Technologies

- JavaScript
- Node.js / Express
- HTML / CSS

## Lancer le projet

1. npm install
2. node server.js
3. Ouvrir index.html
